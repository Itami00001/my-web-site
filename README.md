# MyWebste

Веб-приложение для отображения и управления информацией об автобусах и маршрутах.

## Описание

MyWebste — это веб-сайт, предназначенный для удобного отображения списка автобусов и их маршрутов. Пользователи могут просматривать, фильтровать и изучать подробную информацию о транспорте.

## Функциональности

*   **Просмотр данных:** Отображение полного списка автобусов и маршрутов.
*   **Фильтрация:** Возможность фильтрации автобусов по различным параметрам (например, по названию или номеру).
*   **Детальная информация:** Просмотр подробной информации о каждом автобусе и его маршруте.

## Технологии

*   **Backend:** Node.js, Express.js
*   **База данных:** PostgreSQL
*   **Пакетный менеджер:** npm или Yarn
*   **Контейнеризация:** Docker, Docker Compose

## Запуск с помощью Docker (Рекомендуемый способ)

Это самый простой способ запустить проект, так как все зависимости уже настроены в контейнерах.

### Предварительные требования

Убедитесь, что на вашем компьютере установлены:
*   [Docker](https://docs.docker.com/get-docker/)
*   [Docker Compose](https://docs.docker.com/compose/install/)

### Быстрый старт

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/Itami00001/my-web-site.git
    cd my-web-site
    ```

2.  **Запустите проект:**
    ```bash
    docker-compose up --build
    ```
    *(Флаг `--build` нужен для первой сборки или после изменения кода)*

3.  **Откройте в браузере:**
    Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000).

    *Примечание: База данных PostgreSQL будет автоматически запущена в отдельном контейнере, и все необходимые таблицы должны быть созданы.*

## Установка и настройка (вручную, без Docker)

Если вы предпочитаете запускать проект напрямую, следуйте этим шагам.

### Предварительные требования

*   [Node.js](https://nodejs.org/) (с npm)
*   [PostgreSQL](https://www.postgresql.org/)

### Шаги по установке

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/Itami00001/my-web-site.git
    cd my-web-site
    ```

2.  **Установите зависимости:**
    ```bash
    npm install
    ```
    *или, если вы используете Yarn:*
    ```bash
    yarn install
    ```

3.  **Настройте базу данных:**
    *   Создайте новую базу данных в PostgreSQL с именем `my_webste`.
    *   Создайте пользователя `my_webste_user` с паролем `my_webste_password` и предоставьте ему права на базу данных `my_webste`.
    *   Импортируйте предоставленную схему базы данных (SQL-файл) или запустите скрипт для создания необходимых таблиц (`buses`, `routes` и т.д.).

4.  **Запустите приложение:**
    ```bash
    node app.js
    ```

5.  **Откройте в браузере:**
    Перейдите по адресу [http://localhost:3000](http://localhost:3000), чтобы начать использование веб-сайта.

## Использование

После запуска сервера откройте главную страницу приложения. Вы сможете:
*   Просматривать список всех автобусов и маршрутов.
*   Использовать поля фильтрации для поиска нужного автобуса по номеру или названию.
*   Нажимать на элементы списка, чтобы увидеть более подробную информацию.
